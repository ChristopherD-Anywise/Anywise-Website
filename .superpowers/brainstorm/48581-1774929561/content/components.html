Design Section 2: Component-Specific Overrides

Targeted rules for elements that can't be handled by variable swaps alone

1. Hero Section

Current (Dark)

  • Night Earth globe texture
  • Dark gradient overlay: rgba(0,0,0,0.92) → rgba(0,0,0,0.25)
  • White heading text, green gradient on italic line
  • Green atmospheric glow shader on globe
  • Renderer clear color: #0a0d0a

Light Mode Changes

  • Globe texture → NASA Blue Marble Day (earth-day.jpg)
  • Overlay gradient → Light: rgba(255,255,255,0.85) → rgba(255,255,255,0.3)
  • Heading text → Dark: var(--text-primary) = #1a1d1a
  • Italic gradient line → Keep green gradient (works on both light/dark)
  • Glow shader → Softer, warm white-green tint instead of bright green
  • Renderer clear color → #f5f6f3 (match --bg-deep)
  • Directional light → Increase intensity, reposition to simulate sun
  • Sub-text → rgba(0,0,0,0.6) instead of rgba(255,255,255,0.7)

2. Navigation

  • Backdrop → rgba(250,251,248,0.85) with blur (currently rgba(10,13,10,0.85))
  • Nav links → Dark text, green hover
  • CTA button → Keep green accent button (works on both)
  • Bottom border → rgba(0,0,0,0.06)

3. Capability Cards

Current

Dark green gradients: linear-gradient(145deg, #1a3d1e, #1b5e2a, #163d1f)

Light Mode

  • Background → White card with subtle green-tinted left border
  • Border → rgba(0,0,0,0.08) + 3px solid var(--accent) on left
  • Spotlight effect → Softer, using rgba(57,168,73,0.06) instead of 0.15
  • Hover → Slight shadow elevation instead of gradient shift
  • Text → Dark text (auto via variables)

4. Stats Bar (Track Record)

  • Background → rgba(255,255,255,0.7) with blur (currently rgba(0,0,0,0.35))
  • Border → rgba(0,0,0,0.08)
  • Stat values → Dark green var(--accent) (keeps contrast on white)
  • Stat labels → rgba(0,0,0,0.5)
  • Dividers → rgba(0,0,0,0.06) (currently rgba(255,255,255,0.06))

5. Product Cards, News Cards, CTA Section

  • Product cards → White bg, subtle border, green-tinted shadow on hover
  • News cards → White bg, image overlay reduced to rgba(0,0,0,0.3)
  • CTA section → Light gray bg (var(--bg-deep)), green radial glow uses rgba(57,168,73,0.06)
  • Credentials marquee → Logos need no change (already work on both), fade edges switch to white gradient
  • Footer → Light gray bg, dark text (auto via variables)

6. JavaScript / Globe Logic

  • Time detection → Get current AEDT time on page load (UTC+11). If 6am–6pm, set document.documentElement.dataset.theme = 'light'
  • Globe texture → Conditional load: day texture for light, night for dark
  • Day texture URL → https://unpkg.com/three-globe@2.41.12/example/img/earth-blue-marble.jpg
  • Renderer clear color → Set based on theme
  • Lighting → Increase directional light intensity to 3.5, reposition to (5, 3, 5) for sun effect
  • Glow shader → Reduce intensity multiplier from 0.6 to 0.3, shift color toward warm white
  • Green point light → Reduce intensity from 0.8 to 0.3 in light mode

Transition: On page load, the JS sets data-theme before first paint. A 0.5s CSS transition on background-color and color on key elements ensures a smooth fade if the theme changes. The globe texture loads asynchronously — the scene renders with correct lighting immediately, texture fills in.
