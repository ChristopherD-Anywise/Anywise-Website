How it fits together

shared.css

CSS custom properties (dark/light), nav, footer, typography, buttons, section dividers — everything that appears on every page.

posts.json

Array of post objects. blog/post.html reads ?slug= from the URL, finds the matching post, and renders it. No new HTML file needed per post.

product pages

Each product gets its own HTML file: hero, how it works, use cases, features, CTA. All link shared.css.

engage/index.html

Standalone page (also openable as a modal overlay from the main nav CTA). Contains the qualified enquiry form.
